# Módulo Marketplace

Pendiente de portar desde `_referencia/camila` y `_referencia/mercanto`.

## Scope
- Directorio de tiendas y productos por distrito
- Restaurant: menús, pedidos, mesas
- Ofertas, reseñas, favoritos
- Mapa con Leaflet

## Estrategia
Fusionar lo mejor de Camila (26 tablas, restaurant, inventario, licencias)
con Mercanto (ofertas, reseñas, mapa) en un solo schema `market`.
Ver `docs/ROADMAP.md` Fase 3 para detalles.
