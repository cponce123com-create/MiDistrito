# Módulo Radar Vecinal

Portado desde `_referencia/radar`. Contiene reportes ciudadanos, alertas de pánico, personas desaparecidas y anuncios.

## Backend
- `routes/` — Rutas Express del módulo
- `services/` — Lógica de negocio
- `schema/` — Tablas Drizzle en schema `radar`
- `permissions/` — Permisos del módulo

## Frontend
- `components/` — Componentes React específicos
- `pages/` — Páginas del módulo
- `api/` — Llamadas API específicas
- `hooks/` — Hooks React

## Dependencias
- Core: `@midistrito/db`, `@midistrito/api-zod`
- Ninguna dependencia de otro módulo
